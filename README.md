# クラウドランナー 🪖

兵士を率いて迫り来る敵を撃破するブラウザ型アクションゲーム。

## 🎮 遊び方

- **マウス左右移動** / **スマホスワイプ** で部隊を操作
- 兵士が自動で前方に向かって射撃
- **ゲート**に弾を当てて獲得 → 兵士数を増やす
- **🔥連射** / **💥拡散** 武器アイテムをゲットして火力UP
- 敵が自分の位置に到達したら **GAME OVER**
- 中ボス・大ボスは HP が 0 になるまで撃ち続けろ！

## ✨ 機能

| 機能 | 内容 |
|------|------|
| 🎵 BGM | Web Audio API による手続き型BGM（ノーマル／ボスフェーズ） |
| 🔊 SE | 射撃・ヒット・撃破・ダメージ・ゲート獲得音 |
| 📳 振動 | iPhone Vibration API 対応（ヒット・撃破・ダメージ） |
| 🏆 スコア | 撃破ボーナス・ゲート獲得・クリアボーナス・ハイスコア保存 |
| 🎯 弾幕 | 東方Project風 6パターン弾幕（照準・扇・全方位・螺旋・壁・十字） |
| 🤖 敵AI | 通常敵がプレイヤーを回避しながら左右移動 |
| 📱 PWA | オフライン動作・ホーム画面追加対応 |

## 🚀 起動方法

```bash
# ブラウザで直接開く（サーバー不要）
open index.html

# またはローカルサーバー
python3 -m http.server 3456
# → http://localhost:3456/
```

## 🏗 技術スタック

- **言語**: Vanilla JavaScript + HTML5 Canvas 2D
- **音声**: Web Audio API（AudioContext, OscillatorNode, GainNode）
- **PWA**: Service Worker + Web App Manifest
- **振動**: Vibration API
- **永続化**: localStorage（ハイスコア）
- **ビルド不要**: 単一HTMLファイル完結

## 🎭 ゲームシステム

### スコア
- 雑魚敵撃破: `100 × 最大HP`
- 中ボス撃破: `2,000pt`
- 大ボス撃破: `10,000pt`
- フェーズ2突入ボーナス: `500pt`
- ゲート獲得: `200pt`
- クリアボーナス: `1,000pt + 兵士数 × 50pt`

### 弾幕パターン（東方風）
1. **照準弾** – プレイヤーを追う
2. **扇形弾** – 前方扇状に展開
3. **全方位リング** – 12〜18方向全弾
4. **螺旋弾** – 回転しながら展開
5. **弾幕壁** – 横一列・隙間あり
6. **回転十字弾** – 4方向ずつ3段
